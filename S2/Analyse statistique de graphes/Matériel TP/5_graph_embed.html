On va maintenant lancer DeepWalk. Les paramtres sont (entre autres) :

  • walk_number : le nombre de marches alatoires (Default is 10) ;
  • walk_length : la longueur de chaque marche alatoire (Default is 80) ;
  • dimensions : la dimension de l'espace dans lequel on plonge les donnes (Default is 128) ;
  • window_size: la taille de la fentre qui dfinit le voisinage de chaque noeud (Default is 5).

# plot nodes graph
def plot_nodes(embedding,node_no):
    X = embedding[node_no]
      
    pca = PCA(n_components=2)
    pca_out= pca.fit_transform(X)
      
      
    plt.figure(figsize=(15,10))
    plt.scatter(pca_out[:, 0], pca_out[:, 1])
    for i, node in enumerate(node_no):
        plt.annotate(node, (pca_out[i, 0], pca_out[i, 1]))
    plt.xlabel('Label_1')
    plt.ylabel('label_2')
    plt.show()

Node2vec¶

On rappelle que Node2vec est une version amliore de DeepWalk, dans laquelle la marche alatoire est biaise. La page officielle de Node2Vec est ici https://github.com/eliorc/node2vec On va suivre l'exemple en dmo.

Commencez par installer node2vec dans un terminal avec pip install node2vec ou depuis le notebook (mais ce n'est pas recommand) et importez la librairie.

On va ensuite tester la mthode sur le graphe de similarit G obtenu ci-dessus.

Les options de Node2Vec sont similaires celles de DeepWalk

  • dimensions: Embedding dimensions (default: 128)
  • walk_length: Number of nodes in each walk (default: 80)
  • num_walks: Number of walks per node (default: 10)
  • p: Return hyper parameter (default: 1)
  • q: Inout parameter (default: 1)

Si vous laissez les valeurs de p,q par dfaut, vous retombez normalement sur la mthode DeepWalk, donc a n'a pas d'intrt. Mais on ne sait pas forcment comment choisir ces paramtres. (Note: In node2vec paper they searched the [0.25,4]*[0.25,4] grid and it did have a significant impact on performance, for the task of multi-label node classification.)

Notez que contrairement DeepWalk :

  • la taille de la fentre window n'est pas un paramtre du modle mais de la fonction d'ajustement aux donnes fit
  • les donnes se passent dans le modle (parce que les probas de la marche alatoire dpendent des voisinages).
