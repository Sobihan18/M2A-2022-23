Preliminaries ¶

Let us focus on the compressed sensing problem: finding a sparse vector $x \in \mathbb R^d$ such that $y = Ax$, where the vector of measurements $y$ and the sensing matrix $A$ are given.

For this purpose, we will assess different strategies:

  • a greedy approach: orthogonal matching pursuit;
  • solving basis pursuit with an interior point method (scipy.optimize.linprog);
  • solving a regularized problem: $\operatorname{minimize}_{x \in \mathbb R ^d} \|x\|_1 + \frac{\mu}{2} \|Ax - y\|_2^2$, with proximal methods;
  • solving basis pursuit with Douglas-Rachford method;
  • introducing a new (total variation) regularization in order to improve reconstruction.

In this practical session, we are provided with a measurement function, that returns three components.

The first two are of interest for now.

A greed algorithm: orthogonal matching pursuit ¶

In this section, we aim at providing a a feasible point to the problem

$$ \begin{array}{cl} \displaystyle{ \operatorname{minimize}_{x \in \mathbb R ^d} } & \displaystyle{ \|Ax - y\|_2 } \\ \operatorname{st} & \displaystyle{ \|x\|_0 \le s } \end{array} $$

with orthogonal matching pursuit.

Question 1.

Define a function omp(y, A, s=None, eps=None), that implements orthogonal matching pursuit and that:

  • returns an $s$-sparse approximation of x;
  • returns the sequence $(\|Ax_k-y\|_2)_k$;
  • terminates when $x_k$ is $s$-parse or when $\|Ax_k-y\|_2 \le \epsilon$ ($\epsilon > 0$ given).

You can use a set object for the set of indexes: index = set(), index.add(a_new_index), A[:, list(index)] for the columns of interest.

Plot the approximation error and the solution provided by orthogonal matching pursuit.

In [5]:
# Answer
def omp(y, A, s=None, eps=None):
    n = A.shape[1]  # Size of the signal to recover = x
    if not s and not eps:
        eps = 1e-3
        s = n
    elif not s:
        s = n
    else:
        eps = 0
    
    x = np.zeros(n)  # Initial x
    r = -y  # Residue
    err = [np.linalg.norm(r)]  # L2 error ||Ax-y||_2 = ||r||_2
    ind = set()  # Support of x
    while np.linalg.norm(x, ord=0) < s and err[-1] > eps:
        ind.add( np.argmax(np.abs(A.T.dot(r))) )
        x = minL2(y, A[:, list(ind)])
        r = A[:, list(ind)].dot(x) - y
        err.append( np.linalg.norm(r) )
    
    xsol = np.zeros(n)
    xsol[list(ind)] = x
    return xsol, err

Basis pursuit ¶

We aim here at solving the basis pursuit problem:

$$ \begin{array}{cl} \displaystyle{ \operatorname{minimize}_{z \in \mathbb R ^{2d}} } & \displaystyle{ \sum_{i=1}^{2d} z_i } \\ \operatorname{st} & \left\{ \begin{array}{l} \displaystyle{ [A; -A]z = y } \\ \displaystyle{ z_i \ge 0, \forall i \in [2d] } \end{array} \right. \end{array} $$

with an off-the-shelf simplex method (scipy.optimize.linprog).

A regularized problem: proximal methods ¶

In this section, we decide to approximate the linear constraint with a regularization.

Therefore, we aim at solving: $$ \operatorname{minimize}_{x \in \mathbb R ^d} \|x\|_1 + \frac{\mu}{2} \|Ax - y\|_2^2, $$ where $\mu > 0$.

Question 1.

Show that: $$ [\operatorname{prox}_{\gamma \|\cdot\|_1} (x) ]_i = \max \left(0, 1- \frac{\gamma}{|x_i|} \right) x_i. $$

Implement this function with signature proxL1(x, gamma).

Answer: the soft-thresholding operator is given in Example 2.24. Just check that this statement is equivalent to Example 2.24.

In [9]:
# Answer
def proxL1(x, gamma):
    return np.fmax(0, 1 - gamma / np.fmax(1e-15, np.abs(x))) * x

Question 2.

Define a function ista(y, A, mu=1., n_it=100), that:

  • performs a proximal gradient descent with fixed step size;
  • terminates after n_it iterations;
  • returns an approximate solution $x$ and the sequence of objective values $(\|x_k\|_1 + \frac{\mu}{2} \|Ax_k - y\|_2^2)_k$.

Run this function to recover the original image.

Plot the image, the objective function and compute the error $\|Ax - y\|_2$.

In [10]:
# Answer
def ista(y, A, mu=1., n_it=100):
    Q = A.T.dot(A)
    Ay = A.T.dot(y)
    
    gamma = 1 / (mu * np.linalg.eigvalsh(Q).max())
    x = np.zeros(A.shape[1])
    
    obj = []
    for it in range(n_it):
        x = proxL1(x - gamma*mu*(Q.dot(x) - Ay), gamma)
        obj.append( np.linalg.norm(x, ord=1) + mu/2*np.linalg.norm(A.dot(x)-y)**2 )

    return x, obj

# Answer
def fista(y, A, mu=1., n_it=100):
    Q = A.T.dot(A)
    Ay = A.T.dot(y)
    
    gamma = 1 / (mu * np.linalg.eigvalsh(Q).max())
    x = np.zeros(A.shape[1])
    z = np.zeros(A.shape[1])
    m = 0
    
    obj = []
    for it in range(n_it):
        x_new = proxL1(z - gamma*mu*(Q.dot(z) - Ay), gamma)
        m_new = (1 + np.sqrt(1 + 4*m**2))/2
        z = x_new + (m-1)/m_new * (x_new - x)
        x = x_new
        m = m_new
        obj.append( np.linalg.norm(x, ord=1) + mu/2*np.linalg.norm(A.dot(x)-y)**2 )

    return x, obj

The exact problem: Douglas-Rachford method ¶

Conversely to the approximation in the previous section, we aim now at solving the original problem involved in compressed sensing:

$$ \operatorname{minimize}_{x \in \mathbb R ^d} \|x\|_1 + \chi_{\mathcal A}(x), $$

where $\mathcal A = \{x \in \mathbb R^d : Ax = y\}$.

Question 1.

Show that $\operatorname{prox}_{\chi_{\mathcal A}}(x) = (I - A^\top(AA^\top)^{-1}A)x + A^\top(AA^\top)^{-1}y$.

Answer: by definition of the proximal operator, we consider the problem of minimizing $\frac 12 \|u-x\|_2^2 + \chi_{\mathcal A}(u)$ with respect to $u$, that is minimizing $\frac 12 \|u-x\|_2^2$ st $Au=y$. The Lagrangian is $L(u, \lambda) = \frac 12 \|u-x\|_2^2 + \lambda ^\top Au - \lambda ^\top y$. Then, KKT conditions help characterizing the solution:

  1. stationarity for primal variable: $u - x + A^\top \lambda = 0$, that is $u = x - A^\top \lambda$;
  2. primal feasibility: $Au = y$, that is $Ax - AA^\top\lambda = y$ or $\lambda = (AA^\top)^{-1}(Ax-y)$.

Combining 1. and 2. gives $\operatorname{prox}_{\chi_{\mathcal A}}(x) = x - A^\top (AA^\top)^{-1}(Ax-y) = (I - A^\top(AA^\top)^{-1}A)x + A^\top(AA^\top)^{-1}y$.

# Answer
def DRmethod(y, A, n_it=100, version=1):
    n = A.shape[1]
    
    pA = A.T.dot(np.linalg.inv( A.dot(A.T) ))
    AX = np.eye(n) - pA.dot(A)
    Ay = pA.dot(y)
    proxA = lambda x: AX.dot(x) + Ay
    
    if version == 1:
        proxf = lambda x, s: proxL1(x, s)
        proxg = lambda x, s: proxA(x)
    else:
        proxf = lambda x, s: proxA(x)
        proxg = lambda x, s: proxL1(x, s)
    
    z = np.zeros(n)  # Initial point
    mu, s = 1, 1  # Relaxation and step size
    
    obj = []
    for it in range(n_it):
        x = proxf(z, s)
        z = z + mu*(proxg(2*x-z, s) - x)
        obj.append( np.linalg.norm(x, ord=1) )
    return x, obj

Another sparsity prior ¶

In order to improve image reconstruction, we propose to regularize with a total variation operator $T$.

The problem to minimize becomes: $$ \operatorname{minimize}_{x \in \mathbb R ^d}~ \|Tx\|_1 + \frac{\mu}{2} \|Ax-y\|_2^2. $$

This problem can be rewritten in:

$$ \begin{array}{cl} \displaystyle{ \operatorname{minimize}_{z, x} } & \displaystyle{ \|z\|_1 + \frac{\mu}{2} \|Ax-y\|_2^2 } \\ \operatorname{st} & \displaystyle{ Tx = z. } \end{array} $$

We aim at solving this problem with ADMM.

Question 1.

Show that minimizing the augmented Lagrangian with respect to $z$ boils down to: $$ z_{k+1} = \operatorname{prox}_{\frac 1 \gamma \|\cdot\|_1}(Tx_k + \frac 1 \gamma \nu_k). $$

Answer: the augmented Lagrangian is $L_\gamma(z, x, \nu) = \|z\|_1 + \frac{\mu}{2} \|Ax-y\|_2^2 + \nu^\top Tx - \nu^\top z + \frac \gamma 2 \|Tx-z\|_2^2$. Then, $$ \begin{align*} \operatorname{argmin}_z L_\gamma(z, x_k, \nu_k) &= \operatorname{argmin}_z \|z\|_1 - \nu_k^\top z + \frac \gamma 2 \|z\|_2^2 - \gamma z^\top Tx_k \\ &= \operatorname{argmin}_z \frac 1 \gamma \|z\|_1 + \frac 12 \|z - (Tx_k + \frac 1 \gamma \nu_k)\|_2^2 \\ &= \operatorname{prox}_{\frac 1 \gamma \|\cdot\|_1}(Tx_k + \frac 1 \gamma \nu_k). \end{align*} $$

Question 2.

Show that minimizing the augmented Lagrangian with respect to $x$ boils down to: $$ x_{k+1} = (\mu A^\top A + \gamma T^\top T)^{-1}(\mu A^\top y + T^\top(\gamma z_{k+1} - \nu_k)). $$

Answer: stationarity with respect to $x$ gives: $$ \begin{align*} 0 &=\nabla_x L_\gamma(z_{k+1}, x, \nu_k) \\ &= \mu A^\top (Ax-y) + T^\top \nu_k + \gamma T^\top (Tx-z_{k+1}) \\ &= (\mu A^\top A + \gamma T^\top T)x - (\mu A^\top y - T^\top \nu_k + \gamma T^\top z_{k+1}). \end{align*} $$ Therefore, $x_{k+1} = (\mu A^\top A + \gamma T^\top T)^{-1}(\mu A^\top y + T^\top(\gamma z_{k+1} - \nu_k))$.

# Answer
def ADMM(y, A, T, mu=1., n_it=100, gamma=1.):
    n = A.shape[1]
    p = T.shape[0]
    
    Q = np.linalg.inv(mu*A.T.dot(A) + gamma*T.T.dot(T))
    Ay = mu*A.T.dot(y)
    
    x = np.zeros(n)  # Initial point
    z = np.zeros(p)  # Initial point
    nu = np.zeros(p)  # Initial dual point
    
    obj = []
    for it in range(n_it):
        z = proxL1(T.dot(x) + nu/gamma, 1/gamma)
        x = Q.dot(T.T.dot(gamma*z-nu) + Ay)
        nu += gamma*(T.dot(x)-z)
        
        obj.append( np.linalg.norm(T.dot(x), ord=1) + mu/2*np.linalg.norm(A.dot(x)-y)**2 )
    
    return x, obj
